About This Project

MCA Final Year Project — Mohan Babu University, Tirupati 2025-2026

Project Overview

An intelligent computer vision system for real-time yoga posture analysis and corrective feedback. The system employs a marker-less human pose estimation approach using MoveNet to capture body movements from live camera input.

A Multi-Layer Perceptron (MLP) classifier recognizes yoga postures and evaluates alignment, while a rule-based biomechanical decision mechanism analyzes joint angles against predefined thresholds.

TensorFlow.js MoveNet Lightning MLP Classifier Web Speech API Canvas API HTML5 + Vanilla JS

System Pipeline

1

Video Capture

Live webcam input captured via HTML5 MediaDevices API, mirrored for natural interaction

2

MoveNet Pose Estimation

MoveNet Lightning detects 17 body keypoints (nose, shoulders, elbows, wrists, hips, knees, ankles, eyes, ears) at ≤80ms latency

3

Feature Extraction

Keypoint coordinates normalized and joint angles computed using inverse cosine of 3-point vectors for 8 critical joints

4

MLP Classification

Multi-Layer Perceptron neural network (simulated) classifies the current posture from 15 supported yoga poses using 34 input features

5

Biomechanical Validation

Rule-based engine compares detected joint angles against ideal thresholds per pose; generates correction messages per joint

6

Feedback Delivery

Visual feedback on canvas overlay + text corrections + voice guidance via Web Speech API synthesis
